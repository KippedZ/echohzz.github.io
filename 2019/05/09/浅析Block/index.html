什么是Block?

一个对象，封装了自动变量和函数及其上下文。也有另外一种说法——一段可执行的代码。

Block objects are a C-level syntactic and runtime feature. They are similar to standard C functions, but in addition to executable code they may also contain variable bindings to automatic (stack) or managed (heap) memory. A block can therefore maintain a set of state (data) that it can use to impact behavior when executed.

– Apple《Blocks Programming Topics》

__block_impl

我们可以通过搜索找到这个结构体的声明：

这个结构体可以算是Block的真身了，包含一个isa指针（所以我们认为Block是一个对象）。Flags代表标志信息。Reserved是保留字段。还有一个FuncPtr，这是一个函数指针，指向我们Block的函数体。

__main_block_desc_0

这是结构体可以称作是Block的描述文件，主要记录Block需要的内存大小。

再来看下这个构造函数，第一个参数是函数指针。第二个是Block的描述信息。_multiplier是我们在main函数里面声明的局部变量，这里被Block捕获了。

至此，我们已经对Block的本质有了一个基本的了解，做一个简单的总结：
